
# webpack-react

A starter project for those who want pre configured webpack config.


## Installation

Install project with npm

```bash
  git clone https://github.com/amirking59/webpack-react
  
  cd webpack-react
  npm install
```

## development

To run this on local device

```bash
  npm run start
```

## produciton

To build for produciton

```bash
  npm run build
```
